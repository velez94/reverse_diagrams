# Product Overview

**Reverse Diagrams** is a continuous documentation tool that creates architecture diagrams and audits AWS services from their current state. It's designed as a "Documentation as Code" solution.

## Core Purpose
- Generate visual diagrams of AWS Organizations structure
- Create diagrams for AWS IAM Identity Center (SSO) configurations  
- Audit and inspect AWS services through automated diagram generation
- Provide console-based viewing of AWS resource relationships

## Key Features
- **AWS Organizations**: Visualize organizational units, accounts, and hierarchical structure
- **IAM Identity Center**: Map groups, users, and permission assignments
- **Console Viewer**: Interactive terminal-based viewing of configurations
- **Auto-generation**: Automatically create Python diagram code and PNG outputs
- **JSON Export**: Save AWS resource data as structured JSON for further analysis

## Target Users
DevOps engineers, cloud architects, and AWS administrators who need to document and audit their AWS infrastructure automatically.
